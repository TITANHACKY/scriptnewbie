Telegram Chat Bot using Python

Dec 7, 2017
Some of the Important methods:
  • getMe
  • getUpdates
  • sendMessage
getMe:

A simple method for testing your bot's auth token. It returns the basic information about the bot in form of a User object.

https://api.telegram.org/bot<token>/getMe

You will get the basic information like bot id, bot firstname, lastname and username, etc.

getUpdates:

Incoming updates in the bot is stored for 24 hours. You can get the Incoming updates using getUpdates method. You will get a json file as a result.

https://api.telegram.org/bot<token>/getUpdates

Using the above method you will get some informations like message id, sender user id, username, firstname, lastname, message. If the user send a photo then you will get photo id, if it was a video then video id of if it is a docucment then document id.

Before bot gets deleted, you can get your file using file id whenever in your lifetime. But the file id changes for every bot.

sendMessaage:

This method is used to send message to the user. To send a message you need chat_id and the text.

https://api.telegram.org/bot<token>/sendMessage?chat_id=<chat_id>&text=<text>

To format the text you should use the parse_mode. You can use either markdown-style or HTML-style formatting.. To use Formatting use parse_mode=Markdown or MarkdownV2 or html.

https://api.telegram.org/bot<token>/sendMessage?chat_id=<chat_id>&text=<text>&parse_mode=<style>
