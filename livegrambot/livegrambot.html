In this article you'll learn how to build a bot like livegram bot.

What is livegram bot? What is the features?

Livegram bot is a simple chat bot. It forwards the message from the user to the bot owner and if the owner reply to any forwaded message then the bot will send that message to the replied user.

How to build the bot?

Just think the logic. You should get the message from the user and forward it to the owner and if the owner replied to any message you should send the message to the replied user.

If you know how to build a simple chat bot then it is easy. If you don't know then click here

Keep the stpes in mind: 1)getting updates 2)forwarding messages from user to owner 3)getting updates form the owner what he is replying and for whom. 4)Sending message to the user.

Always don't run the script alone. Also try to understand it.
